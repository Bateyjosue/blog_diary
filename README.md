# 📖 [Catlog] <a name="about-project"></a>

> blog_daily is a Rails web app, a classic example of a blog website. A fully functional website that will show the list of posts and empower readers to interact with them by adding comments and liking posts.

 Here is the list of projects that will guide you through the steps described above

- Project 1: Setup and controllers.
- Project 2: Controllers specs.
- Project 3: Creating a data model.
- Project 4: Processing data in models.
- Project 5: Views.
- Project 6: Forms
- Project 7: Validations, Model specs, and n+1 problems.
- Project 8: Add Devise.
- Project 9: Add authorization rules.
- Project 10: Integration specs for Views.
- Project 11: Add API endpoints.
- Exercise: API documentation.

### Prerequisites

### Setup

Clone this repository:

```sh
  git clone https://github.com/Bateyjosue/blog_diary.git
```
Go to your project

```sh
  cd blog_diary
```

### Install

*Important! You need to have ruby and rspec installed in your computer*

Install the project dependencies with:

```sh
  bundle install
```

### Usage

To run the project, execute the following command:

```sh
  ruby main.rb
```


### Run the project

- To run the project, run the following command:

```sh
  rails server or  rails s
```

- To Setup the db run

```sh
  rails db:create
```
- See Results by navigating to

```sh
  https://localhost:3000
```
